# Projet Master 1 – Analyse de données sur le cytomégalovirus

## Contexte
Ce dépôt présente un projet réalisé dans le cadre de ma première année de master. Il porte sur l’analyse et la visualisation de données cliniques liées au cytomégalovirus (CMV) à l’aide de Python.

## Objectif
L’objectif du projet était de nettoyer les données, recoder certaines variables, puis produire des représentations graphiques permettant d’explorer plusieurs caractéristiques de la cohorte étudiée.

## Méthodes
Le travail a été réalisé en Python avec principalement :
- pandas
- numpy
- matplotlib
- seaborn

## Analyses réalisées
- préparation et recodage des variables
- analyse descriptive
- boxplots
- graphiques en barres empilées
- visualisation des associations entre variables cliniques

## Cadre académique
Projet réalisé dans le cadre d’un enseignement de première année de master, avec une approche orientée data analysis et visualisation biomédicale.
